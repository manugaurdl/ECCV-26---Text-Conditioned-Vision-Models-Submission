In summary, we make the following contributions:
\begin{enumerate}
\item We introduce \model{} (Steerable ViT), a new paradigm for multimodal modelling that equips any pretrained ViT with text steerability via a simple, scalable pretext task in grounding, adding only X\% new parameters.
\item We demonstrate a Pareto improvement over existing approaches: \model{} steers visual features with text while preserving the base ViT's downstream transfer, bridging the gap between unsteerable unimodal encoders and quality-degraded multimodal models.
\item We find that text prompts serve as a zero-shot generalization mechanism, enabling \model{} to transfer to novel domains -- personalized object discrimination, industrial anomaly detection -- matching or outperforming dedicated methods and supervised fine-tuning baselines, without task-specific training, across multiple ViT backbones.
\end{enumerate}
