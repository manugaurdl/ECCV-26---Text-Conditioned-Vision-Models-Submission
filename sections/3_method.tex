SHIFT injects text conditioning into a frozen Vision Transformer through sparse, gated cross-attention layers that steer visual features without degrading them.
As illustrated in Figure~\ref{fig:arch}, our architecture consists of three components: (1) a frozen RoBERTa-Large text encoder that produces token-level embeddings from the input prompt, (2) a lightweight multimodal adapter that projects these text embeddings into the visual embedding space, and (3) tanh-gated cross-attention layers interleaved within a frozen ViT that progressively fuse textual cues into the visual representation.
Given an image and a text prompt, SHIFT produces text-contextualized patch-level visual features that remain in the base ViT's embedding space.
We train the model on a diverse mixture of referential segmentation datasets (Section~\ref{sec:data}) using a binary segmentation objective at the ViT patch level (Section~\ref{sec:objective}).
The segmentation head serves solely as a training signal and is discarded at inference; the output of interest is the steered visual representation itself.

\label{sec:arch}

\textbf{Visual Encoder.}
We use a frozen DINOv2 ViT-B/14~\cite{oquab2024dinov2} as our visual backbone.
For an input image $X_v \in \mathbb{R}^{H \times W \times 3}$, the ViT produces a sequence of $N$ patch tokens $Z_v \in \mathbb{R}^{N \times d}$ along with a \texttt{[CLS]} token, where $d$ is the embedding dimension.
All original ViT parameters remain frozen throughout training; new capacity enters exclusively through the interleaved cross-attention layers described below.

\textbf{Text Encoder.}
We adopt a frozen, pretrained RoBERTa-Large~\cite{liu2019roberta} as the text encoder.
Given an input prompt $X_t$, the encoder extracts a variable-length sequence of token embeddings $Z_t \in \mathbb{R}^{L \times d_t}$, where $L$ is the number of text tokens and $d_t$ is the text embedding dimension.
We $\ell_2$-normalize these embeddings before passing them to the multimodal adapter.

\textbf{Multimodal Adapter.}
A trainable two-layer MLP projects the normalized text embeddings $Z_t$ into visual-aligned tokens $H_t \in \mathbb{R}^{L \times d}$, matching the ViT's embedding dimension $d$.
We add learnable positional encodings to $H_t$ to preserve token order before injecting them into the ViT through the gated cross-attention mechanism.

\textbf{Gated Cross-Attention Layers.}
To fuse textual guidance into the ViT's residual stream, we adapt the gated cross-attention dense formulation from Flamingo~\cite{alayrac2022flamingo}.
Each inserted layer applies multi-head cross-attention followed by a feed-forward network (FFN), where the visual patch tokens $Z_v^{(\ell)}$ at layer $\ell$ serve as queries and the adapted text tokens $H_t$ serve as keys and values:
\begin{equation}
    \hat{Z}_v^{(\ell)} = \text{CA}(Z_v^{(\ell)}, H_t) = \text{softmax}\!\left(\frac{Q K^\top}{\sqrt{d_k}}\right) V, \quad Q = Z_v^{(\ell)} W_Q, \; K = H_t W_K, \; V = H_t W_V.
\end{equation}
The cross-attention output is integrated into the residual stream through a tanh gate with a layer-specific learnable scalar $\alpha_\ell$, initialized to zero:
\begin{equation}
    Z_v^{(\ell+1)} = Z_v^{(\ell)} + \tanh(\alpha_\ell) \cdot \text{FFN}\!\left(\hat{Z}_v^{(\ell)}\right).
    \label{eq:gated_ca}
\end{equation}
Since $\tanh(0) = 0$, the model is mathematically identical to the frozen ViT at initialization.
As training progresses, the gates learn to open, allowing the model to smoothly incorporate text conditioning.

We insert these cross-attention layers at every third transformer block starting from block 2 (i.e., blocks 2, 5, 8, 11 for the 12-block ViT-B).
The final transformer block always remains unmodified, ensuring that the output features stay close to the pretrained ViT's representation space.

\subsection{Training Objective}
\label{sec:objective}

We train SHIFT on binary referential segmentation at the ViT's patch resolution rather than at the pixel level.
Given an image $X_v$ and a referential expression $X_t$ describing a single target object, the model predicts which patches overlap the referred object.

The ViT partitions the image into an $n \times n$ grid of non-overlapping patches.
We tokenize the ground-truth binary segmentation mask $M \in \{0, 1\}^{H \times W}$ into patch-level labels $\hat{y} \in [0, 1]^{n \times n}$ by computing the fraction of foreground pixels within each patch.
A linear classification head maps each patch token $z_i \in \mathbb{R}^{d}$ to a scalar logit $p_i$.
We optimize the network with a binary cross-entropy loss over all patches:
\begin{equation}
    \mathcal{L} = -\frac{1}{n^2} \sum_{i=1}^{n^2} \left[ \hat{y}_i \log \sigma(p_i) + (1 - \hat{y}_i) \log (1 - \sigma(p_i)) \right],
    \label{eq:loss}
\end{equation}
where $\sigma$ denotes the sigmoid function.
This objective forces the cross-attention layers to route task-relevant textual information into the corresponding visual patch tokens, producing steered representations as a byproduct.

\subsection{Training Data}
\label{sec:data}

We train on a mixture of referential segmentation and grounding datasets to ensure diversity in both visual domains and textual expression styles:

\textbf{RefCOCO, RefCOCO+, RefCOCOg}~\cite{kazemzadeh2014referitgame, yu2016refcocog} provide referring expressions grounded in COCO images.
RefCOCO+ excludes spatial language (e.g., ``left of''), encouraging the model to rely on appearance cues, while RefCOCOg contains longer, more descriptive expressions that exercise the model's capacity for detailed textual understanding.

\textbf{Visual Genome}~\cite{krishna2017visual} (preprocessed following MDeTr~\cite{kamath2021mdetr}) contributes region descriptions paired with bounding boxes across densely annotated scenes.
These descriptions span a broader vocabulary and more complex spatial relationships than the RefCOCO family, increasing the diversity of text conditioning signals.
We use SAM2 to convert bounding boxes to binary segmentation masks.

\textbf{Mapillary Vistas}~\cite{neuhold2017mapillary} provides street-level imagery with fine-grained panoptic annotations.
Including this dataset expands the visual domain beyond COCO's indoor and object-centric bias toward outdoor, driving, and urban scenes, improving out-of-distribution generalization. We use Describe Anything's synthetically generated referential expressions along with accompanying binary segmentation masks. 

This combination exposes the model to varied scene complexities (from single objects to dense urban panoramas), expression lengths (from two-word labels to multi-sentence descriptions), and visual domains (indoor, outdoor, street-level), encouraging robust text-conditioned representations.
