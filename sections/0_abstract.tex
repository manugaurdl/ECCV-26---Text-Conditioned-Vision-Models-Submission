

\begin{abstract}
Pretrained Vision Transformers (ViTs) such as DINOv2 and MAE provide generic image features that can be applied to a variety of downstream tasks such as retrieval, classification, and segmentation.  
However, such representations tend to focus on the most prominent object in the image, and so, are difficult to adapt to other visual concepts.
% \MG{\sout{Multimodal models such as CLIP  and MLLMs}} 
Multimodal LLMs can be tailored to textual prompts, but the resulting representations tend to be language-centric and lose their effectiveness for generic visual tasks.
To address this, we introduce
% \MG{Steerable ViT?}
\textbf{\model{}}, a framework that equips any ViT with the ability to steer both its global and local visual representations with natural language.
While most multimodal models tend to late-fuse visual encoders with text features (e.g., CLIP), \model{} \textit{early}-fuses textual information into layers of the visual encoder using lightweight cross-attention.
We demonstrate that our steerable visual features can focus on arbitrary objects in an image while preserving 
% \MG{\sout{the ability to perform generic visual tasks}}
the underlying representation quality, demonstrated by
fine-grained retrieval, classification, and segmentation performance.
% \MT{fine-grained retrieval is not generic visual task?}
\model{} also matches or outperforms dedicated approaches
% \MG{\sout{on zero-shot}}
on anomaly detection and personalized object discrimination, exhibiting zero-shot generalization to out-of-distribution domains without task-specific training.
\vspace{-2mm}
\keywords{Visual Representation \and Semantic Steering \and Text-Conditioning}
\vspace{-13mm}
\end{abstract}
