\begin{abstract}
Pretrained Vision Transformers (ViTs) such as DINOv2 and MAE provide generic image features that can be applied to a variety of downstream tasks such as retrieval, classification, and segmentation.  
However, such representations tend to focus on the most prominent object in the image, 
% and so, are difficult to adapt to other visual concepts.
\MG{and cannot be steered toward specific visual concepts of interest.}
% \MG{\sout{Multimodal models such as CLIP  and MLLMs}} 
Multimodal LLMs can be guided with textual prompts,
% \MG{can be steered with natural language,}
but the resulting representations tend to be language-centric and lose their effectiveness for generic visual tasks.
To address this, we introduce
% \MG{Steerable ViT?}
% \textbf{\model{}}, a framework 
\textit{Steerable Visual Representations}, a new class of visual representations, whose global and local features can be steered with natural language.
% While most multimodal models tend to late-fuse visual encoders with text features (e.g., CLIP), \model{} \textit{early}-fuses textual information into layers of the visual encoder using lightweight cross-attention. 
While most vision-language models (e.g., CLIP) fuse text with visual features after encoding (\textit{late} fusion), we inject text directly into the layers of the visual encoder (\textit{early} fusion) via lightweight cross-attention. 
% We demonstrate that our steerable visual features can focus on arbitrary objects in an image while preserving 
% % \MG{\sout{the ability to perform generic visual tasks}}
% the underlying representation quality, demonstrated by
% fine-grained retrieval, classification, and segmentation performance.
We introduce benchmarks for measuring \textit{representational steerability}, and demonstrate that our steerable visual features can focus on 
% \JR{\st{arbitrary}
any desired objects in an image while preserving the underlying representation quality.\textbf{}
Our method also matches or outperforms dedicated approaches
% \MG{\sout{on zero-shot}}
on anomaly detection and personalized object discrimination, exhibiting zero-shot generalization to out-of-distribution tasks.
