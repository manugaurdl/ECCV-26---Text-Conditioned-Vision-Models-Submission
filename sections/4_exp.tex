\subsection{Baselines}
We benchmark SHIFT against a number of existing methods that allow for prompt-based textual steering of their internal vision representations as well as the (potentially unimodal) base models underlying our method. 

For \textbf{late-fusion dual-encoders} (e.g., CLIP~\cite{radford2021_clip}, SigLIP~\cite{zhai2023_siglip}), images and prompts are embedded independently using their respective encoders, limiting multimodal interaction to a simple post-hoc fusion, where the resulting global or dense image embeddings are combined with the text representations via element-wise addition. 
As with the considered \textbf{vision-only baselines}, global feature aggregation is conducted as intended by the base method (e.g., CLS or mean pooling). FLAIR~\cite{xiao2025flair} produces localized image-level representations by applying text-conditioned attention pooling on vision tokens.
We extract dense text-conditioned vision representations from \textbf{MLLMs} by concatenating the user prompt $t$ followed by the image $I$ into a multimodal input sequence and selecting the last-language-block states at image-token positions. For global features, we adapt the last-token pooling approach of E5-V~\cite{Jiang2024E5VUE} and append a postfix instructing the model to summarize the image contents in one word while focusing on the aspect specified by $t$.
Text-conditioned representations are obtained from \textbf{open-vocabulary grounding and segmentation methods} via the modality-fused encoder states, ignoring decoder components responsible for generating bounding boxes or segmentation mask. 
Specifically, with SAM2~\cite{ravi2024sam2}, we take the patch-level outputs of the \textit{Fusion Encoder} and apply mean-pooling to get image-level features. 
Similarly, we use the outputs of the cross-modal \textit{Feature Enhancer} layer of GroundingDINO~\cite{liu2023groundingdino}. However, as features are available across multiple spatial scales, we first interpolate them to a common intermediate feature scale before being averaged to get dense pseudo-patch-level features.

Where supported by the underlying architecture, we process images at a resolution of $336\times336$. For models with non-configurable input dimensions, we still resize to this resolution prior to processing to ensure comparable levels of visual information.

To validate that text conditioning steers the global feature vector, we introduce a conditional retrieval benchmark built on the SUN scene dataset.
We select six scene classes split into two categories: indoor (bathroom, kitchen, living room) and outdoor (park, residential area, street), each containing 500 images.
Using the Flux image editing model, we inpaint five scene-relevant objects into 100 images per class (e.g., a mug in a kitchen, a vase in a living room).
The task is one-vs-all retrieval: given a query image containing inpainted object $A$, retrieve a same-class image that also contains $A$ from the remaining 499 images.
Random chance is approximately 20\%, since 99 out of 499 candidates share the query object.
For text-steerable models, retrieval is conditioned on the inpainted object class (e.g., ``mug'').
Successful retrieval under text conditioning implies that the model shifts the global representation from the scene-level concept (``kitchen'') to a text-contextualized one (``mug in kitchen'').

\subsubsection{Query-Agnostic Encoders Collapse to Salient Concepts}

Query-agnostic encoders fail at conditional retrieval because their features collapse to the dominant scene concept (Figure~\ref{fig:cond_ret_baselines}).
MAE barely exceeds random chance, and DINOv2 roughly doubles MAE's accuracy owing to its more object-centric representations.
CLIP and SigLIP, despite operating in a multimodal embedding space, are unable to contextualize their visual features through post-hoc late fusion with text.
In contrast, our method achieves 96.6\% and 91.3\% accuracy on indoor and outdoor scenes respectively, confirming that text conditioning shifts the global representation from the scene level (``kitchen'') to the queried concept (``mug in kitchen'').

Conditioning on a random object class instead of the correct one causes retrieval performance to degrade catastrophically, demonstrating that the steerability is indeed driven by the text query rather than by an unconditional feature improvement.

Figure~\ref{fig:umap-neighbourhood} illustrates this on a ``residential neighbourhood'' class via UMAP: DINOv2 embeddings show no separability across inpainted object types, whereas our text-conditioned embeddings form distinct clusters for each queried concept.

\subsubsection{Late Fusion Does Not Enable Steerability}
A natural follow-up is whether post-hoc fusion with text can rescue query-agnostic representations.
For CLIP and SigLIP, which already share a multimodal embedding space, we fuse visual features with text embeddings via element-wise addition.
This yields a negligible boost of 0.02\%, confirming that late fusion cannot steer frozen representations.
Learning query-aware features within the ViT layers (early fusion) is essential.

\subsubsection{MLLMs: Steerable but Inefficient}

MLLMs can steer visual features, but at substantial computational cost.
InternVL3-1B scores only 51.2\%; scaling the LLM to 2B parameters raises this to 77\%.
Our method, built on the same visual encoder (SigLIP), outperforms their 1B and 2B variants by 40\% and 14\% respectively while adding only 130M parameters via cross-attention blocks.
Our approach can be viewed as an inverted MLLM: instead of conditioning a language model on frozen visual features (late fusion), we condition the visual features on text within the ViT layers themselves (early fusion).
Because text shapes the visual representation directly, it can guide attention toward task-relevant regions in a way that post-hoc fusion cannot.
Recent work~\cite{chen2025vlm_benchmark} corroborates this, showing that VLMs perform substantially worse than their underlying visual encoder on vision-centric benchmarks.

\subsubsection{Open-Vocabulary Localization Models}
GroundingDINO and SAM2 are highly steerable, with SAM2 nearly matching our retrieval accuracy.

Text conditioning routes self-attention to tokens corresponding to the queried object, improving dense predictions for non-salient concepts.
In the previous section, we validated that text steers global semantics; here we examine whether it also controls where the model attends at the patch level.
We construct a mosaic benchmark by stitching together four object-centric images from PASCAL-VOC into a single composite, reducing the saliency of each individual object.
This setup lets us quantify the foreground bias of self-attention and measure how effectively text redirects it.

\subsubsection{Qualitative Analysis}
DINOv2's \texttt{[CLS]} attention maps concentrate on salient foreground objects, confirming the saliency bias hypothesized in Section~\ref{sec:intro}.
Figure~\ref{fig:steer_attn} (right) illustrates this on a mosaic: DINOv2 allocates attention toward the house and bathtub but ignores the tree along the road.
When conditioned on ``tree,'' our method routes attention to the full extent of the tree and partly to a potted plant in an adjacent quadrant due to its semantic association.
Additional qualitative examples appear in the Appendix.

\subsubsection{Quantitative Evaluation}
We quantify each encoder's ability to localize distinct objects by using \texttt{[CLS]} attention maps as a proxy for spatial focus.
We compute the area under the ROC curve (AUC) between attention maps and ground-truth segmentation masks from PASCAL-VOC for each queried object.
As shown in Figure~\ref{fig:steer_attn} (left), text-conditioned features outperform query-agnostic features by 33.7\% AUC on this dense prediction task.

GroundingDINO and SAM2 also route attention toward queried objects (Figure~\ref{fig:mosaic_baselines}), which is expected given that they are optimized for localization.
However, while these models satisfy the first two desirable properties (global steerability and dense attention routing), they fail to preserve representation quality of the frozen visual features as illustrated by Figure~\ref{fig:teaser}.
We analyze this tradeoff in the next section.

\subsection{Feature Quality vs.\ Steerability: A Pareto Analysis}
\label{subsec:pareto}

Steerability must not come at the cost of feature quality.
The primary purpose of visual representation learning is to produce features that transfer to downstream tasks such as classification and segmentation.
A trivial solution that completely overwrites visual features with text yields total steerability but discards the underlying representation.

Existing models fall into three regimes (Figure~\ref{fig:teaser}):
SAM2 and GroundingDINO achieve high steerability but produce task-specific representations that score poorly on standard vision probes;
MLLMs perform reasonably on classification probes but falter on dense tasks and require billions of parameters for moderate steerability;
standard encoders (DINOv2, SigLIP) produce rich transferable features but are not steerable.
Our approach bridges this gap, achieving high steerability with only 10\% degradation on classification and segmentation probes.

\subsubsection{The Gate as a Continuous Steerability Knob}

The tanh-gated cross-attention mechanism (Eq.~\ref{eq:gated_ca}) provides a continuous control knob for the quality-steerability tradeoff.
Because the gates are zero-initialized, the model is mathematically identical to DINOv2 at initialization (gate factor = 0.0).
Through training, the gates learn to open, allowing text features to progressively steer the visual representation.
At inference, we can scale the gate magnitude to smoothly interpolate between the original DINOv2 subspace and a fully text-conditioned state.

Plotting this trajectory (the green curve in Figure~\ref{fig:teaser}) reveals a clear Pareto frontier.
We observe an optimal operating point at a scaling factor of 0.6: the model preserves the foundational feature quality of DINOv2 while unlocking high text steerability, outperforming both standard encoders and models like SAM2 and GroundingDINO.

Figure~\ref{fig:gate_factor} (right) further shows that increasing the gate factor from 0.0 to 0.6 doubles the mosaic benchmark AUC from 0.15 to 0.34, confirming that the gate directly controls how much attention is routed through text-relevant tokens.

\subsection{Text Specificity Controls Semantic Granularity}
\label{subsec:text_density_exp}
\begin{figure}[t]
  \centering
  \includegraphics[width=0.98\textwidth]{figures/eccv_pods.png}
  \caption{\textbf{Text specificity controls feature granularity.} On the PODS benchmark, coarse prompts (supercategory) degrade performance relative to DINOv2 by discarding instance-level cues. Class-level prompts improve over the baseline. Fine-grained descriptive prompts surpass even a supervised fine-tuned DINOv2, confirming that detailed text acts as a precise steering mechanism.}
  \label{fig:pods}
\end{figure}

The semantics of the steered representation are highly sensitive to the specificity of the text prompt.
In Section~\ref{subsec:cond_ret_exp}, we observed that conditioning on a random object class halves retrieval performance; here we explore this sensitivity systematically.
We evaluate on the Personal Object Discrimination Suite (PODS), which tests encoding of personalized objects under distribution shifts.
The model does not simply ``add'' information to the DINOv2 baseline; rather, it steers the feature space to match the resolution of the prompt.

\subsubsection{From Coarse to Fine-Grained Steering}
We analyze performance across three levels of textual detail (Figure~\ref{fig:pods}):
\begin{itemize}
  \item \textbf{Coarse (Supercategory):} Conditioning on high-level supercategories discards lower-level texture and shape cues that are irrelevant to the superclass but crucial for instance identity. This explains the performance drop relative to vanilla DINOv2, which retains these features by default.
  \item \textbf{Intermediate (Class):} Conditioning on class names, which capture category-level attributes, significantly improves performance over the baseline.
  \item \textbf{Fine-grained (Descriptive):} The largest gains occur with highly descriptive text queries (generated manually or synthetically). The model performs instance-level discrimination in a zero-shot manner, surpassing a supervised fine-tuned DINOv2. Detailed prompts act as a precise steering mechanism.
\end{itemize}

This demonstrates that natural language can steer features effectively in out-of-distribution domains where labeled data for fine-tuning is scarce.

\subsection{Visualizing the Steered Embedding Space}
\label{sec:tsne_viz}

t-SNE visualizations reveal how text conditioning reshapes the embedding space topology.
Standard pretrained encoders like DINOv2 exhibit a fixed topology dictated by saliency bias, clustering images by the dominant foreground object.
Our method dynamically reconfigures this topology based on the input text query.
We explore this through two steering scenarios: \textit{Hierarchical Semantic Steering} and \textit{Compositional Attribute Steering}.


\subsubsection{Hierarchical Semantic Steering}

This experiment tests whether text can steer clustering from fine-grained object classes to broad semantic superclasses.

\paragraph{Setup.} We select images containing $N$ distinct object classes divided into $k$ supercategories.

\paragraph{Analysis.} 
As shown in Figure~\ref{fig:tsne_semantic}, the baseline DINOv2 embeddings (left) form six distinct clusters, rigidly adhering to granular object identity and treating ``dog'' and ``horse'' as separate concepts.

Conditioning our model on \texttt{``Animal''} (right) transforms the embedding space: all animal classes (dog, cat, horse, bird, sheep) collapse into a single macro-cluster, while non-animal classes are pushed away.
Crucially, object-level semantics are preserved within the animal cluster: dogs and cats remain closer neighbors than birds (Figure~\ref{fig:tsne_semantic}), confirming that text controls the granularity of clustering without destroying fine-grained structure.

\subsubsection{Compositional Attribute Steering}

This experiment tests a more challenging scenario: clustering visually dissimilar objects (e.g., office chair and bike) based on a shared sub-object attribute (wheel), ignoring their primary semantic category.

\paragraph{Setup.} We curate a heterogeneous dataset split into two compositional groups:
\begin{itemize}
    \item \textbf{Has-Part ``Wheel'':} Cart, wheelchair, bike, car, airplane.
    \item \textbf{No-Part ``Wheel'':} Apple, house, person.
\end{itemize}

\paragraph{Analysis.} 
DINOv2 fails to capture this shared attribute: the t-SNE visualization (Figure~\ref{fig:tsne_compositional}, left) shows clusters organized by object identity, with a wheelchair clustered closer to a person (contextual bias) than to a cart.

Conditioning on \texttt{``Wheel''} produces a binary separation of the embedding space (Figure~\ref{fig:tsne_compositional}, right): one cluster containing all objects with wheels, regardless of whether they are vehicles or medical equipment, and another containing objects without wheels.
This confirms that text conditioning can route attention to specific compositional attributes rather than the global object concept.
